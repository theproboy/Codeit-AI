
# Codeit-AI 

Codeit-AI is an AI-powered online code compiler and intelligent coding assistant that allows users to write, run, debug, and analyze code in real time using a secure Python backend and a modern web interface.

##  Features (Planned)
- Online Python code compiler
- Custom input/output support
- Secure code execution
- AI-based code explanation
- Error detection and suggestions
- Web-based editor (VS Code style)
